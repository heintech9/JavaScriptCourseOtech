Understanding Their Roles in Building Web Pages.

HTML: Noun (Defines the structure and content of the webpage)

CSS: Adjective (Describes the appearance and style of the HTML elements)

JavaScript: Verb (Adds behavior and interactivity to the webpage)

ES5 vs ES6

Comparison of ES5 and ES6 Features.

Feature ES5 ES6
Variable Declarations var let, const
Functions Regular functions Arrow functions
Classes Constructor functions and prototypes class syntax
Strings Concatenation with + Template literals (` `` `)
Destructuring No built-in destructuring { a, b } = obj, [x, y] = arr
Modules No native modules import, export
Asynchronous Operations Callbacks Promise
Parameters No default values Default parameters, Rest and Spread operators
Object Literals No shorthand syntax Shorthand properties and methods

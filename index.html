Journey to React

Core JavaScript Skills

  • Variables and Data Types
    • Understanding let, const, and var
    • Familiarity with primitive and complex data types
  • Functions
    • Declaration and expression syntax
    • Arrow functions
    • Higher-order functions and callbacks
  • Control Structures
    • Conditional statements (if, else, switch)
    • Loops (for, while, forEach)
  • ES6+ Features
    • Destructuring assignment
    • Spread and rest operators
    • Template literals
    • Default parameters
    • Modules (import and export)
  • Objects and Arrays
    • Creating, manipulating, and iterating over objects and arrays
    • Array methods like map, filter, reduce, etc.
  • DOM Manipulation
    • Basic understanding of how to select and manipulate DOM elements
    • Event handling

Advanced JavaScript Concepts

  • Asynchronous JavaScript
    • Promises and async/await
    • Fetch API and AJAX
  • Closures and Scope
    • Understanding lexical scope and closures
    • Context (this keyword)
  • Error Handling
    • Try/Catch
    • Custom errors

Tooling and Ecosystem

  • Package Managers
    • npm and yarn: installing and managing dependencies
  • Module Bundlers
    • Basic understanding of Webpack, Parcel, or similar tools
  • Version Control
    • Git basics: cloning, committing, pushing, and pulling
  • Development Environment
    • Setting up a code editor (like VSCode) with essential extensions
    • Using linters (ESLint) and formatters (Prettier)
